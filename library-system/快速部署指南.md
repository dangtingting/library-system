# 图书馆管理系统 - 5分钟快速部署指南

## 🎯 目标
在5分钟内部署一个功能完整的图书馆管理系统，无需编程知识，一键完成。

## 📋 前提条件

### 必需软件（请提前安装）
1. **Java 8 或更高版本**
   - 下载地址：https://www.oracle.com/java/technologies/downloads/
   - 验证安装：打开命令提示符，输入 `java -version`

2. **MySQL 5.7 或更高版本**
   - 下载地址：https://dev.mysql.com/downloads/mysql/
   - 安装时记住设置的root密码
   - 验证安装：确保MySQL服务正在运行

## ⚡ 一键部署（3步完成）

### 第1步：解压项目
将 `library-management-system.zip` 解压到任意目录，例如：
```
D:\projects\
```

### 第2步：修改数据库密码（如需要）

**如果MySQL的root密码是 `123456`，跳过此步骤**

如果密码不是 `123456`，请修改配置文件：

1. 打开文件：
   ```
   library-management-system\src\main\resources\application.properties
   ```

2. 找到并修改这一行：
   ```properties
   spring.datasource.password=123456
   ```
   
   改为你的MySQL密码：
   ```properties
   spring.datasource.password=你的密码
   ```

### 第3步：运行部署脚本

1. **右键点击** `deploy.bat`
2. 选择 **"以管理员身份运行"**
3. 等待脚本自动完成（约2-3分钟）
4. 看到 "部署完成！" 提示即可

## 🚀 启动系统

### 方式一：使用启动脚本（推荐）
1. 在部署目录中找到 `start-library-system.bat`
2. 双击运行
3. 等待系统启动（约30秒）
4. 看到 "Started LibraryManagementSystemApplication" 表示启动成功

### 方式二：使用命令行
```bash
cd library-management-system
cd target
java -jar library-management-system-1.0.0.jar
```

## 📖 快速使用

### 添加图书
1. 用管理员账号登录
2. 点击"图书管理"
3. 点击"添加图书"
4. 填写图书信息，点击"添加"

### 借阅图书
1. 点击"图书管理"
2. 找到要借阅的图书
3. 点击"借阅"按钮
4. 输入用户ID，确认借阅

### 归还图书
1. 点击"借阅管理"
2. 找到要归还的图书记录
3. 点击"归还"按钮

## 🛠️ 管理工具

### 重置数据库
如果数据出现问题：
1. 运行 `reset-database.bat`
2. 重新启动系统
3. 所有数据将恢复到初始状态

### 修改端口号
如果8080端口被占用：

1. 打开 `start-library-system.bat`
2. 找到这一行：
   ```bash
   java -jar "%PROJECT_DIR%\target\library-management-system-1.0.0.jar" --server.port=8080
   ```
3. 修改端口号：
   ```bash
   java -jar "%PROJECT_DIR%\target\library-management-system-1.0.0.jar" --server.port=8081
   ```

## 🔧 常见问题速查

### 问题1：deploy.bat 运行失败
**原因**：没有以管理员身份运行
**解决**：右键 → 以管理员身份运行

### 问题2：MySQL连接失败
**原因**：密码错误或MySQL未启动
**解决**：
1. 检查MySQL服务是否启动
2. 修改 `application.properties` 中的密码

### 问题3：8080端口被占用
**解决**：
1. 修改端口号（见"修改端口号"部分）
2. 或关闭占用8080的程序

### 问题4：系统无法访问
**检查清单**：
- [ ] Java已安装
- [ ] MySQL服务已启动
- [ ] 项目已成功构建
- [ ] 浏览器访问 http://localhost:8080
