## 🚀 快速开始（3步部署）

### 第1步：准备工作
确保已安装：
- Java 8+
- MySQL 5.7+
- MySQL服务已启动

### 第2步：一键部署
1. 解压项目文件
2. 右键点击 `deploy.bat`
3. 选择"以管理员身份运行"
4. 等待部署完成（约2-3分钟）

### 第3步：启动系统
1. 运行 `start-library-system.bat`
2. 打开浏览器访问: http://localhost:8080
3. 使用管理员账号登录

## 💡 核心功能

### 📚 图书管理
- 图书信息录入、编辑、删除
- 多维度搜索（书名、作者、ISBN）

### 👥 用户管理

## 🛠️ 技术特点

### 后端
- **Spring Boot 2.7.18** - 快速开发框架
- **Spring Data JPA** - 数据访问
- **MySQL 8.0** - 关系型数据库
- **Maven** - 项目构建

### 前端
- **HTML5 + CSS3 + JavaScript** - 现代化前端
- **响应式设计** - 适配多种设备
- **AJAX** - 异步数据交互
- **Bootstrap风格** - 美观UI

### 数据库
- **3个核心表** - books, users, borrow_records
- **2个统计视图** - 图书借阅统计、用户借阅统计
- **索引优化** - 提升查询性能

## 📋 常见问题

### Q1: 部署失败怎么办？
**A**: 检查是否以管理员身份运行，确保Java和MySQL已安装。

### Q2: 数据库连接失败？
**A**: 修改 `application.properties` 中的数据库密码。

### Q3: 8080端口被占用？
**A**: 修改启动脚本中的端口号。

### Q4: 如何重置数据？
**A**: 运行 `reset-database.bat` 脚本。
