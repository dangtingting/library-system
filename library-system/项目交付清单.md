## 🚀 部署方式

### 方式一：一键部署（推荐）
```bash
# 右键点击，以管理员身份运行
deploy.bat
```

### 方式二：手动部署
```bash
# 1. 创建数据库
mysql -uroot -p < library-management-system/src/main/resources/database/init.sql

# 2. 构建项目
cd library-management-system
mvnw.cmd clean package -DskipTests

# 3. 运行项目
java -jar target/library-management-system-1.0.0.jar
```

### 方式三：IDEA开发环境
```bash
# 1. 用IDEA打开项目
# 2. 修改数据库配置
# 3. 运行 LibraryManagementSystemApplication.java
```
